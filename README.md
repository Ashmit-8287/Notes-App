# 📝 Notes App

A modern and responsive Notes App built using React.js and Tailwind CSS.

## 🚀 Features
- Add Notes
- Delete Notes
- Responsive UI
- Dynamic State Management
- Interactive Card Layout
- React Hooks
- Tailwind CSS Styling

## 🛠️ Technologies Used
- React.js
- JavaScript
- Tailwind CSS
- HTML5
- CSS3

## 📚 Learning Outcome
This project helped me improve my understanding of React state management, event handling, and frontend development.
